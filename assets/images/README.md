# Image Assets

## Directory Structure

- `hero/` - Hero section background images
- `marathons/` - Marathon event photos and memories
- `routes/` - Running route scenery and location photos
- `icons/` - Social media and UI icons

## Image Guidelines

### Hero Images
- Recommended size: 1920x1080px or larger
- Format: JPG (optimized for web)
- Examples needed:
  - `runner-landscape.jpg` - Main hero background
  - `runner-sunrise.jpg` - Sunrise running scene
  - `profile-running.jpg` - Personal profile photo while running

### Marathon Photos
- Size: 1200x800px minimum
- Format: JPG
- Examples:
  - `sydney-2024-finish.jpg`
  - `gold-coast-2024.jpg`
  - `melbourne-2023.jpg`
  - `canberra-2024.jpg`
  - `first-marathon.jpg`
  - `medal-collection.jpg`

### Route Photos
- Size: 1200x800px minimum
- Format: JPG
- Examples:
  - `bondi-beach.jpg`
  - `bondi-sunrise.jpg`
  - `tamarama-view.jpg`
  - `bronte-pool.jpg`
  - `centennial-park.jpg`
  - `centennial-trees.jpg`
  - `centennial-pond.jpg`
  - `centennial-morning.jpg`
  - `sydney-harbour.jpg`
  - `harbour-bridge.jpg`
  - `opera-house-view.jpg`
  - `circular-quay.jpg`

## Image Optimization

Before uploading images:
1. Resize to appropriate dimensions
2. Compress using tools like TinyPNG or ImageOptim
3. Target file size: < 500KB per image
4. Use progressive JPG for better loading experience
