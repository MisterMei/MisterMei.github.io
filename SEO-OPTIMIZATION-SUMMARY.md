### Step 2: Deploy to GitHub Pages
```bash
git add .
git commit -m "Add comprehensive SEO optimization"
git push origin main
```

Wait 2-5 minutes for GitHub Pages to rebuild.

### Step 3: Verify Deployment
Visit your site and check:
- [ ] All pages load correctly
- [ ] No broken links
- [ ] Images display properly

### Step 4: Test SEO Implementation

**Test Structured Data:**
1. Go to [Google Rich Results Test](https://search.google.com/test/rich-results)
2. Enter your homepage URL
3. Check for errors (should see Person schema)
4. Test other pages (about, marathons, routes)

**Test Social Sharing:**
1. [Facebook Debugger](https://developers.facebook.com/tools/debug/)
   - Enter your URL
   - Click "Scrape Again" if needed
   - Verify image and description appear

2. [Twitter Card Validator](https://cards-dev.twitter.com/validator)
   - Enter your URL
   - Verify card preview looks good

**Test Mobile:**
- [Mobile-Friendly Test](https://search.google.com/test/mobile-friendly)

### Step 5: Submit to Search Engines

#### Google Search Console
1. Go to [search.google.com/search-console](https://search.google.com/search-console)
2. Click "Add Property"
3. Enter your full URL
4. Verify ownership (HTML tag method):
   - Copy verification meta tag
   - Add to `<head>` of index.html
   - Click "Verify"
5. Submit sitemap:
   - Go to "Sitemaps" section
   - Enter: `sitemap.xml`
   - Click "Submit"

#### Bing Webmaster Tools
1. Go to [bing.com/webmasters](https://www.bing.com/webmasters)
2. "Add a site"
3. Import from Google Search Console (easiest)
   OR verify manually
4. Submit sitemap.xml

### Step 6: Monitor Results

Check Google Search Console weekly for:
- Indexing status (all 4 pages indexed)
- Search queries bringing traffic
- Click-through rates
- Any errors or warnings
