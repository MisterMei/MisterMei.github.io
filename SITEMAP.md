# Running Journal - Sitemap

## Site Structure

```
/
├── index.html              # Homepage with introduction, featured runs, recent updates
├── marathons.html          # Marathon records (full & half)
├── routes.html             # Regular running routes with GPX maps
├── about.html              # Personal story and running journey
│
├── css/
│   ├── main.css           # Core styles
│   ├── responsive.css     # Mobile/tablet breakpoints
│   └── animations.css     # Subtle transitions
│
├── js/
│   ├── language.js        # Bilingual toggle functionality
│   ├── main.js            # General interactivity
│   └── map.js             # GPX route visualization
│
├── assets/
│   ├── images/
│   │   ├── hero/          # Hero background images
│   │   ├── marathons/     # Marathon event photos
│   │   ├── routes/        # Route scenery photos
│   │   └── icons/         # Social media icons
│   ├── audio/
│   │   └── ambient.mp3    # Background music (optional)
│   └── gpx/
│       └── *.gpx          # Route GPS data files
│
└── README.md              # Deployment instructions
```

## Page Breakdown

### Homepage (index.html)
- Hero section with scenic runner image
- Personal introduction
- Running statistics overview
- Recent milestones
- Social media links
- Featured routes preview

### Marathons (marathons.html)
- Full marathon records table
- Half marathon records table
- Personal best times
- Event photos and memories
- Timeline view

### Routes (routes.html)
- Regular running route cards
- GPX map integration
- Route details (distance, elevation, difficulty)
- Photo galleries
- Embedded videos

### About (about.html)
- Personal running journey
- Why I run
- Running philosophy
- Training approach
- Contact information

## Navigation
- Fixed top navigation bar
- Language toggle (EN/中文) in top-right
- Mobile hamburger menu for small screens
- Smooth scroll navigation
